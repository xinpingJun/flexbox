flex 布局实例

以下属性设置在容器上

  • flex-direction 项目的排列方式

    起点默认x轴左端 .flex-row {flex-direction: row;} 起点默认y轴上沿 .flex-column {flex-direction: column; 反方向 .flex-row-reverse {flex-direction: row-reverse;} 反方向 .flex-column-reverse {flex-direction: column-reverse;

  • flex-wrap :如果一条轴线排不下，如何换行。

    换行.flex-wrap {flex-wrap: wrap;} 不换行.flex-nowrap {flex-wrap: nowrap;

  • justify-content:属性定义项目在x轴上如何对齐。

    两端对齐.space-between {justify-content: space-between;} 左对齐 .justify-start {justify-content: flex-start;} 右对齐.justify-end {justify-content: flex-end;} 水平居中.justify-center {justify-content: center;}

  • align-items :属性定义项目在Y轴上如何对齐。

    垂直居中.align-middle {align-items: center;} 左上角.align-start {align-items: flex-start;} 左下角 .align-end {align-items: flex-end;} 展开.stretch {align-items: stretch;}

  • 水平垂直居中 .flex-center

    .flex-center { justify-content: center; align-items: center; }

以下属性设置在项目上面

  • order : 定义排序

    .flex-first {order: -1;} .flex-last {order: 1;}

  • flex : 子元素分配空间

    .flex-auto {flex: auto;} .flex-none {flex: none;}

  • align-self :属性允许单个项目有与其他项目不一样的对齐方式

    .selft-stretch {align-self: stretch;} .align-self-start {align-self: flex-start;} .align-self-end {align-self:flex-end;} .align-self-middle {align-self: center;}
